# X / Twitter Event Marketing Playbook for AGI Summit SF 2026

Created: July 7, 2026
Use case: Reusable resource for generating X posts that drive ticket sales, replies, reposts, and saves for AGI Summit or similar AI events.

---

## 1. Core principle

The best X post for event ticket sales should not feel like a flyer pasted into a feed. It should feel like a high-signal invitation to a room people do not want to miss.

For AGI Summit, the winning angle is:

> **This is the highest-density AI room in San Francisco: frontier labs, Stanford/Berkeley researchers, founders, investors, and operators in one place.**

The post should create three reactions:

1. **FOMO:** “I should probably be there.”
2. **Status:** “Important AI people will be there.”
3. **Action:** “Tickets are easy to find, and there is a discount code.”

## 2. Current X-specific learnings

These are the practical rules to follow when asking ChatGPT to write X posts for this event.

### Organic post copy

X’s own business guidance favors concise, conversational post copy with a clear call-to-action. It also recommends avoiding hashtags in post copy, avoiding all-caps copy, and using media to help a post stand out.

**Use this in practice:**

- Keep the main post short enough to be instantly understood.
- Use one sharp hook in the first line.
- Do not add a pile of hashtags.
- Do not write the entire post in all caps.
- Add one engagement prompt at the end.
- Use the visual to carry the event details, not the text alone.

### Organic creative

X guidance says media can help a post stand out, but images should not be overloaded with text. For video, keep it very short and use a sound-off strategy when dialogue matters.

**Use this in practice:**

- Use a single premium visual for the main post.
- Keep the visual readable on mobile.
- Use one big headline, one credibility row, one CTA bar.
- Do not cram speaker headshots, every logo, every topic, and every sponsor onto one image.
- For X, a strong single image is usually cleaner than a cluttered carousel.

### Ticket sales and paid ads

For lower-funnel ticket sales, X’s Website Conversions campaigns are designed to optimize toward conversion actions on your website. X also says advertisers should implement X Pixel or Conversion API so campaigns can measure and optimize conversion events.

**Use this in practice:**

- Organic post = engagement + awareness + social proof.
- Pinned first reply = ticket link + discount code.
- Paid post = direct ticket link, Website Conversions objective, X Pixel / Conversion API, purchase or checkout event tracking.
- Test 3–5 creative/copy variants per paid ad group.
- Use urgency and consistency between copy, visual, and landing page.

### Link placement

For organic posts, the recommended tactic is:

1. Main post: no link, strong hook, strong visual, clear “Tickets below ↓”.
2. First reply: ticket link + discount code.
3. Pin the first reply.

Why: a no-link main post can look less like an ad and invite more replies/reposts. However, always test this against putting the link directly in the main post, because link behavior and algorithmic effects on X can change.

For paid conversion ads, put the link in the ad itself and use conversion tracking.

## 4. Best organic X post structure

Use this structure when the goal is both engagement and ticket sales.

```text
[Sharp FOMO hook]

[Authority proof / who is in the room]
[Scale proof]
[Date + venue]

[Event name]

Tickets below ↓

[Engagement question]
```

### Why this works

- The first line sells the emotion.
- The authority proof tells people why the room matters.
- The scale proof makes the event feel real.
- The “Tickets below” line keeps the main post clean.
- The question encourages replies and tagging.

## 7. Visual strategy for X

The visual should stop the scroll in one second. It should not look like a generic conference flyer.

### Recommended specs

- Format: PNG or JPG
- Size: 1200 × 675 px
- Aspect ratio: 16:9 landscape
- Use case: single-image X post
- Mobile rule: headline must be readable when viewed on a phone

### Visual concept that worked for this campaign

A cinematic, high-contrast AGI Summit poster.

**Background:** dark navy / black futuristic gradient with subtle blue-purple glow. Use a stylized premium illustration of the Palace of Fine Arts at night, stage-light beams, and a packed AI conference crowd silhouette in the foreground.

**Main headline:**

```text
THE AI ROOM
YOU CAN’T MISS
```

**Subheadline:**

```text
AGI SUMMIT SF 2026
```

**Proof strip:**

```text
15,000+ ATTENDEES  •  200+ SPEAKERS  •  500+ VCs
```

**Authority line as text chips, not logos:**

```text
OpenAI  ·  Anthropic  ·  Stanford  ·  Berkeley
```

**Bottom CTA bar:**

```text
July 18–19 · Palace of Fine Arts, SF
agisummit.ai · 15% OFF: GenAI-26
```

### Design rules

- Do not use hashtags on the image.
- Do not use official logos unless the user has approved/logo rights.
- Use organization names as text chips instead.
- Do not include speaker headshots unless the user provides approved headshots.
- Do not overfill the visual with 20+ names.
- Use one emotional headline, one proof row, one CTA bar.
- Keep the design premium, cinematic, and modern.

## 9. Ticket-sales playbook

### Organic workflow

1. Post the main copy with the single visual.
2. Immediately reply with the ticket link and discount code.
3. Pin that reply.
4. Have the official account reply to early commenters with real answers, not generic “thanks!” responses.
5. Encourage speakers, partners, and sponsors to quote-post with their own reason to attend.
6. Repost high-quality attendee/speaker replies.
7. Track which hook gets replies, not just likes.

### Paid workflow

1. Use Website Conversions objective for ticket sales.
2. Install X Pixel or Conversion API before launching.
3. Track lower-funnel events such as checkout, purchase, lead, or registration.
4. Use UTM parameters on ticket links.
5. Start with 3–5 creative variants:
   - FOMO room visual
   - speaker/authority visual
   - founder/investor visual
   - agenda/topic visual
   - discount/urgency visual
6. Test different CTAs:
   - “Get tickets”
   - “Save your seat”
   - “Join the room”
   - “Use code GenAI-26”
7. Retarget site visitors, ticket-page visitors, and people who engaged with prior AGI Summit posts.
8. Keep paid copy shorter than organic copy when possible.

## 12. What not to do

Avoid these patterns on X:

- Do not post a generic “Join us for an exciting event!” announcement.
- Do not use five hashtags at the end.
- Do not make the image a dense flyer.
- Do not lead with discounts; lead with value and status.
- Do not list every speaker in the main post.
- Do not put all event details in the copy and then repeat all of them in the image.
- Do not use fake scarcity unless it is true.
- Do not overpromise “the most important AI event in history” unless you can substantiate it.
- Do not use speaker/company logos without permission.
